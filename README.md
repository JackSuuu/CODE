# CODE

A personal collection of programs, exercises, and projects accumulated while learning various programming languages and computer science concepts.

## Languages & Topics

| Directory | Language | Contents |
|-----------|----------|----------|
| `C/` | C | Pointers, memory allocation, data structures, system programming |
| `C++/` | C++ | OOP, data structures, REST API, game dev (snake, spinning cube) |
| `Python/` | Python | Algorithms, AI/ML, GUI, database, OOP, game dev (pygame) |
| `Java/` | Java | Algorithms (Sedgewick), OOP, Spring Boot, Snake game |
| `Haskell/` | Haskell | Functional programming, IO monad, UoE tutorial series |
| `Web-Dev/` | HTML/CSS/JS/TS | CS50 Web, Three.js, TypeScript, React |
| `Swift/` | Swift | iOS app — cyber-todoist |
| `R/` | R | Statistical modelling — Peak Electricity Demand (GB) |
| `MIPS_assembly/` | MIPS ASM | Branching, arrays, sorting, arithmetic |
| `LeetCode/` | Python | LeetCode problem solutions |
| `Jupyter_notebook/` | Python | Data exploration notebooks |
| `Docker/` | Docker | Containerisation intro |
| `PHP/` | PHP | PHP basics |
